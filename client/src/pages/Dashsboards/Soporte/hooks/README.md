# Hooks Directory

Este directorio contiene hooks personalizados específicos para el módulo de Soporte.

## Estructura recomendada:

- `useSoporteData.js` - Hook para gestionar datos de soporte
- `useTickets.js` - Hook para operaciones con tickets
- `useSoporteStats.js` - Hook para estadísticas de soporte
- etc.

Los hooks deben seguir las convenciones de React y ser reutilizables.
